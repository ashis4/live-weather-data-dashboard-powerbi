# 🌦️ Weather Data Dashboard (Power BI)

## 📌 Overview
This project is a real-time weather dashboard built using Power BI by integrating a weather API. It provides live weather updates along with forecasts for up to 3 days, enabling dynamic analysis and visualization of weather patterns.

## 🚀 Features
- Real-time weather data using API integration
- Live dashboard displaying current conditions and forecasts for up to 3 days
- Interactive visualizations for temperature, humidity, and weather trends
- Dynamic updates for data-driven insights
- Implemented API-based data fetching for up-to-date dashboard performance

## 🛠️ Tools & Technologies
- Power BI  
- Weather API  
- Data Transformation  
- Data Visualization  

## 📊 Insights
- Identified patterns and trends in weather data
- Enabled data-driven understanding through visual analytics
- Improved interpretation of real-time environmental data
